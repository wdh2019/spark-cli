# spark-cli

测试项目：用node构建一个可以构建前端项目的cli

## 运行

如果你是windows系统

```bash
pnpm run test-win-g
```

如果你是mac系统

```bash
pnpm run test-mac-g
```

## cli构建项目的步骤

创建项目文件夹

创建模版html文件index.html

创建入口文件main.js

创建package.json

安装依赖


## 提示

本cli的所有关键代码都在bin目录下
